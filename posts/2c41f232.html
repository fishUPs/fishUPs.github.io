1、安装 Node.js 环境： 因为 Hexo 是基于Node.js 的博客框架，就像 Java 要依赖 JDK 环境一样。

  • node下载地址：http://nodejs.cn/download/

2、安装版本控制工具 Git ： 用来将本地项目托管到码云，所以还需要自己注册一个码云的账号

  • 下载地址：https://git-scm.com/download

3.检查是否安装成功
